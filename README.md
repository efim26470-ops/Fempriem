# Fempriem — восстановленная исходная версия

Это архив с исходным вариантом дизайна: Apple/glass-оформление, 8 программ, поиск, скидки, сроки, контакты, PWA-установка и адаптация под ПК/телефон.

## Деплой на GitHub Pages
1. Распакуйте архив.
2. Загрузите файлы из папки в корень репозитория `Fempriem`.
3. Проверьте, что в корне лежат `index.html`, `manifest.webmanifest`, `sw.js`, `icon.svg`.
4. Подождите обновления GitHub Pages.
