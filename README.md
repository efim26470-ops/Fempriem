# Fempriem — финальная тёмная версия

Эта сборка сделана так, чтобы GitHub Pages открывал только новый тёмный сайт, без старой светлой/недоделанной версии.

## Что важно сделать при деплое

1. Открой репозиторий `Fempriem` на GitHub.
2. Лучше удалить старые файлы, которые остались от прошлых сборок: `README.txt`, `app.js`, `data.js`, `styles.css`, `стили.css`, старые `icon-*`, `иконка-*`, `maskable-*`, `манифест.webmanifest`, `закрепленная вкладка Safari.svg`.
3. Загрузи файлы из этого архива в корень репозитория.
4. Убедись, что в корне лежит именно новый `index.html`.
5. После публикации открой сайт с добавкой `?v=dark-final-20260627`, чтобы сбить кэш браузера: `https://efim26470-ops.github.io/Fempriem/?v=dark-final-20260627`.

## Почему раньше открывались разные версии

В репозитории одновременно лежали файлы от нескольких сборок, а браузер и старый service worker могли отдавать закэшированный `index.html`. В этой версии CSS и JS встроены прямо в `index.html`, а `sw.js` чистит старый кэш и отключает старую PWA-копию.
