# Fempriem — обновление для GitHub Pages

Статический сайт справочника приёмной комиссии ФЭМ СПбГТИ(ТУ) на 2026 год.

## Деплой
1. Распакуйте архив.
2. Загрузите содержимое папки в корень репозитория `Fempriem`.
3. В GitHub Pages выберите ветку `main` и папку `/root`.
4. Откройте `https://efim26470-ops.github.io/Fempriem/`.

## Файлы
- `index.html` — страница сайта.
- `styles.css` — адаптивный дизайн.
- `app.js` — фильтры, калькулятор, тема, PWA.
- `data.js` — данные программ.
- `manifest.webmanifest`, `sw.js`, `icon.svg` — PWA и иконка.

## Источники
Основные данные взяты с официального портала ФЭМ: https://gtifem.ru/abiturientam/
