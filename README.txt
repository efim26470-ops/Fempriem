Разместите все файлы и папку icons в корне GitHub Pages-репозитория.
PWA и офлайн-режим работают только при открытии сайта по HTTPS, а не напрямую как file://.
На iPhone: Safari → Поделиться → На экран «Домой».
На macOS Safari: Файл → Добавить в Dock.
В Chrome/Edge: кнопка установки в адресной строке или меню браузера.
